sync.Once в Go используется для гарантии, что определенный 
код будет выполнен только один раз, даже если он вызывается 
из нескольких горутин. Это полезно для инициализации ресурсов, 
которые должны быть созданы только один раз, например, 
создание синглтона или инициализация глобальных переменных.
